Welcome to Bost

Bost is a one-page website hosting platform to use for completely free.

Easy to Use

Bost is very easy to use. All you do is click the create button at the top, paste in HTML (or even just text), assign an account (you don't need to), then upload it!

Open Source

Bost is completely open source on GitHub! You can check the source code out here.

Don't know how to write HTML?

You can just write text! Bost will detect if it's text or not and put it on the appropriate template.
